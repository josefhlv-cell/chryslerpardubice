  5. API VRSTVA
================================================================================

---- src/api/partsAPI.ts ----
KLÍČOVÉ FUNKCE:
- searchParts(query, page, filters) — lokální DB + externí katalog
- searchByCategory(term, page, filters) — hledání podle kategorie
- searchEPC(brand, category, model, sub, page) — EPC kategorie
- decodeVIN(vin) — NHTSA API dekódování
- getRecommendations(part) — podobné díly
- downloadCSV(parts) — export výsledků
- enrichWithSupersessions(parts) — obohacení o náhrady dílů
- mapToPartResult(item, source) — normalizace dat
- applyFilters(parts, filters) — client-side filtrování

TYPY:
- PartResult: id, name, oem_number, price_*, category, family, segment, manufacturer, availability, superseded_by, supersedes
- SearchFilters: brand, model, year, motor, category, manufacturer, minPrice, maxPrice, availability
- PAGE_SIZE = 20

---- src/api/serviceAPI.ts ----
KLÍČOVÉ FUNKCE:
- calculateServiceDue(vehicle, plan) — výpočet urgence servisu
  - effective_km = current_km - km_start
  - effective_last = last_service_km - km_start
  - due when: effective_km - effective_last >= interval
- getVehicleServiceReport(userId, vehicleId) — report pro jedno vozidlo
- getAllVehicleReports(userId) — reporty pro všechna vozidla

TYPY:
- VehicleWithService: id, brand, model, year, engine, vin, current_mileage, km_start
- ServiceDueItem: plan, urgency (due/soon/ok), effectiveKm, kmRemaining, daysRemaining, progressPercent
- VehicleServiceReport: vehicle, items[], history[]
